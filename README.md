# Craigui - Craies Scolaires de Qualité en Guinée

Site vitrine pour Craigui, fabricant et distributeur de craies scolaires en République de Guinée.

🏭 **Fabrication locale | 💰 **Prix imbattables | 🚚 **Livraison partout**

## Stack Technique

- React 18 + Vite
- Tailwind CSS
- React Router
- Dark Mode

## Installation

```bash
npm install
npm run dev
```

## Déploiement

Le site est déployé automatiquement sur Vercel à chaque push sur GitHub.
